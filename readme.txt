2. JavaScript Validation Logic
=================================

Recommendation Page:
- The selections are evaluated using predefined rules.
- Restaurants are recommended based on dietary preferences, budget and purpose.

Registration Page:
- Username must be at least 5 characters and contain only letters, numbers, and underscores.
- Email address must be valid.
- Phone number must be between 8 and 15 digits, with digits only.
- Password must be at least 10 characters, including uppercase, lowercase, numbers, and special characters.
- Password confirmation must match the password field.
- All required fields must be populated before submitting.

Reservation Form:
- Reservation date cannot be in the past.
- Number of guests must be greater than 0.
- Voucher payment displays a voucher code field and hides the credit card field.
- Online payment displays a credit card field and hides the voucher code field.
- Credit card numbers are checked for correct lengths.
    - Via/Mastercard: 16 digits
    - Amex: 15 digits
